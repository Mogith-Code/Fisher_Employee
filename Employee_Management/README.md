# 🐟 Fisher Shop Employee Management - React Client

This directory contains the React front-end application for the Fisher Shop Employee Management system. It is bootstrapped using Vite.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Lint code:
   ```bash
   npm run lint
   ```

## 📁 Source Layout

- **`src/main.jsx`**: Root rendering and initialization.
- **`src/App.jsx`**: Main view routing and logic container.
- **`src/App.css`**: Layout styles.
- **`src/index.css`**: Design tokens, color schemes, and global rules.

For a detailed breakdown of planned features, tech stack details, and overall system design, please refer to the main repository [README.md](../README.md).
